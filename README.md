An interactive and modern **portfolio project** built with **Next.js**, **Framer Motion**, and **shadcn/ui** — designed to demonstrate clean UI, smooth animations, and reusable components.  
Perfect for learning how to build professional-grade portfolios and mastering modern frontend development tools.

## ✨ Features
- ⚡ **Next.js 15** – Hybrid rendering for speed and flexibility  
- 💫 **Framer Motion** – Intuitive, high-performance animations  
- 🧩 **shadcn/ui** – Accessible, composable components with Tailwind CSS  
- 🎨 **Responsive design** – Perfect look across all devices  
- ♻️ **Reusable Components** – Built with scalability in mind  
- 🔒 **TypeScript** – Type-safe and reliable development experience  
- 🧠 **Clean Architecture** – Easy to maintain and extend

## 🛠️ Tech Stack
| Technology | Description |
